# Bridge Demo Script (~3 min) — skeleton

- 0:00–0:20 Problem: short visit, fragmented history, transcript never becomes a usable picture.
- 0:20–1:25 Live first visit (Case A): agent-pulled EMR summary already on screen (PMH, allergies, meds, imaging: none, urgent-care visit, neuro wait time); stream chunks — intake fills what the EMR can't (pattern + progression, phenotype incl. activity worsening + aura, 8-item red-flag screen, functional burden incl. sports + repeat visits, treatment response), PedMIDAS completes, missing questions shrink.
- 1:25–2:05 Clinical clarity: action console, ask one grounded question, click a citation.
- 2:05–2:35 Complete visit → summary + PDF export.
- 2:35–3:00 Toggle Case B follow-up: PedMIDAS trend 32→11, intervention overlay, escalation logic.

Fallbacks: BRIDGE_DEMO_MODE=1 keeps everything deterministic. Never demo with live extraction unless rehearsed.
